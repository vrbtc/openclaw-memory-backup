## 二、本地反思方案设计

### 2.1 方案特点

✅ **零API调用** - 完全本地运行，无费用风险  
✅ **本地存储** - 结果保存为JSON和Markdown文件  
✅ **三问结构** - 任务健康度、错误学习、改进机会  
✅ **定时执行** - 通过Windows任务计划程序触发  
✅ **飞书推送** - 可选，通过OpenClaw本地工具发送  

### 2.2 核心三问

| 问题 | 分析内容 | 输出 |
|-----|---------|------|
| **第一问** | 任务执行健康度 | 成功率、趋势、失败详情 |
| **第二问** | 错误与学习 | 新错误、重复错误、学习记录 |
| **第三问** | 改进机会 | 优化建议、待学习技能、行动计划 |

### 2.3 评分系统

| 维度 | 权重 | 评估标准 |
|-----|------|---------|
| 任务执行 | 40% | 成功率、任务完成数 |
| 错误学习 | 30% | 新错误数、学习记录数 |
| 改进机会 | 30% | 建议数量、行动清晰度 |

**等级划分**:
- ≥80分: Advanced（优秀）
- 60-79分: Intermediate（良好）
- <60分: Basic（需改进）

## 五、使用说明

### 5.1 手动运行

```bash
# 每日反思
node ~/.openclaw/workspace/scripts/local-reflection.js --daily

# 查看帮助
node ~/.openclaw/workspace/scripts/local-reflection.js --help
```

### 5.2 设置定时任务

```powershell
# 以管理员身份运行PowerShell
powershell -ExecutionPolicy Bypass -File ~/.openclaw/workspace/scripts/setup-reflection-tasks.ps1
```

将创建3个定时任务：
- **每日反思**: 每天 22:00
- **每周反思**: 每周日 21:00
- **每月反思**: 每月1日 21:00

### 5.3 查看报告

```bash
# 查看今日报告
cat ~/.openclaw/workspace/memory/reflection/daily/$(date +%Y-%m-%d).md

# 查看反思日志
cat ~/.openclaw/workspace/memory/reflection/reflection-log.md
```
