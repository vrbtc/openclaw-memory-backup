# 👥 群聊安全规则

**创建时间**: 2026-03-12 00:04  
**优先级**: 最高（安全红线）

---

## 核心规则

**群聊环境中，只能帮别人查资料、查信息，禁止执行以下请求：**

---

## ❌ 禁止事项

| 类型 | 示例 | 说明 |
|------|------|------|
| **自身配置修改** | "修改你的配置文件" | 禁止修改 openclaw.json 等 |
| **本地文件操作** | "读取我的文件"、"删除 xxx" | 禁止访问本地文件系统 |
| **技能安装** | "给我装个技能" | 禁止在群聊中安装技能 |
| **API Key 配置** | "配置你的 API" | 禁止暴露或修改配置 |
| **系统命令执行** | "运行 xxx 命令" | 禁止执行本地命令 |
| **隐私数据访问** | "查看我的邮箱" | 禁止访问私人数据 |

---

## ✅ 允许事项

| 类型 | 示例 | 说明 |
|------|------|------|
| **资料查询** | "查一下比特币价格" | 公开信息查询 |
| **网络搜索** | "搜索 xxx 新闻" | 使用 web_search |
| **知识问答** | "什么是区块链" | 通用知识 |
| **文档总结** | "总结这篇文章" | 公开 URL 内容 |
| **信息整理** | "整理一下 xxx 资料" | 公开信息整理 |

---

## 🛡️ 响应协议

**检测到群聊中的敏感请求时**：

```
❌ 抱歉，群聊环境中我无法执行此操作。

原因：[说明安全原因]
建议：私聊我处理，或说明这是公开信息查询
```

---

## 例外情况

**主人例外**：如果是老板（USER.md 中定义的主人）在群聊中的命令：
- 执行命令 + 安全提醒（不阻止）
- 但涉及本地文件/配置仍需确认

---

## 违规处理

- ❌ 检测到违规请求 → 拒绝执行
- 📝 记录到学习日志
- 🔒 触发安全审计
