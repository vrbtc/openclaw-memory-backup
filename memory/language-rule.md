# 🇨🇳 语言使用规则

**创建时间**: 2026-03-11 22:31  
**优先级**: 最高

---

## 核心规则

**所有汇报、沟通、输出必须使用中文，除非以下情况：**

---

## 允许使用英文的情况

| 类型 | 示例 | 说明 |
|------|------|------|
| **专有名词** | Gmail、OpenClaw、BeautifulSoup | 产品名、技能名 |
| **股票代码** | BTC、ETH、AAPL、TSLA | 金融代码 |
| **文件路径** | `C:\Users\Administrator\...` | 系统路径 |
| **代码命令** | `python script.py` | 命令行 |
| **URL 链接** | `https://...` | 网址 |
| **API 字段** | `user_id`、`webhook_url` | 技术字段名 |

---

## 禁止使用英文的情况

| 类型 | 错误示例 | 正确做法 |
|------|---------|---------|
| **任务汇报** | "Task completed" | "任务完成" |
| **状态说明** | "Running"、"Done" | "执行中"、"已完成" |
| **错误提示** | "Permission denied" | "权限拒绝" |
| **日常沟通** | "Please provide..." | "请提供..." |
| **文件描述** | "Core parser code" | "核心解析器代码" |

---

## 违规处理

- ❌ 检测到英文汇报 → 立即纠正
- 📝 记录到学习日志
- 🔄 自我改进，避免再犯

---

## 例外情况

**子 Agent 返回英文结果时**：
1. 总指挥必须翻译成中文
2. 禁止直接转发英文内容
3. 可在括号中保留原文（如需要）

**示例**：
```
✅ 正确：任务完成（Task completed）
❌ 错误：Task completed
```
